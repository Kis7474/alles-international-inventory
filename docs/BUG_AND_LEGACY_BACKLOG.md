## 즉시 수정 완료
1. 인증 없이 앱 첫 화면 접근 가능하던 문제
   - 조치: `/login` 제외 전 경로 세션 선행
2. STAFF가 타 담당자 매입/매출 수정 가능하던 문제
   - 조치: `/api/sales`에 담당자 소유권 강제
3. 시드 재실행 시 비밀번호가 갱신되지 않던 문제
   - 조치: `prisma/seed.ts`의 upsert update에 `passwordHash` 포함
