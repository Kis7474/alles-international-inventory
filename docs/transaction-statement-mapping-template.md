# 거래명세서 PDF/Excel 매핑표 템플릿

아래 표를 그대로 복사해 작성하면, 개발자가 PDF/Excel 양식을 거의 1:1로 구현할 수 있습니다.

## 1) 문서 기본정보

| 항목 | 값(작성 예시) |
|---|---|
| 문서명 | 거래명세서 |
| 버전 | v1.0 |
| 기준 샘플 | 2025-12-거래명세서-샘플.pdf / sample.xlsx |
| 용지 | A4 세로 |
| 기본 폰트 | 맑은 고딕 10pt |
| 여백 | 상 12mm / 하 12mm / 좌 12mm / 우 12mm |
| 허용 오차 | 인쇄 위치 ±2mm |
| 페이지 분할 규칙 | 품목 8행 초과 시 다음 페이지 |

## 2) PDF 레이아웃 매핑표

| 구역 | 표시항목 | 데이터 필드 | 위치/영역(좌표 또는 설명) | 스타일(폰트/크기/정렬/선) | 값 규칙(빈값/줄바꿈/절삭) | 필수 |
|---|---|---|---|---|---|---|
| 헤더 | 제목(거 래 명 세 서) | 고정 텍스트 | 상단 중앙 | Bold, 44~52px, 자간 10~14px | 고정 | Y |
| 좌측 수신정보 | 받는분 | recipientName | 상단 좌측 1행 | 12~14pt | 없으면 '-' | Y |
| 좌측 수신정보 | 참조 | recipientRef | 상단 좌측 2행 | 12~14pt | 없으면 '-' | N |
| 좌측 수신정보 | 전화 | recipientPhone | 상단 좌측 3행 | 12~14pt | 없으면 '-' | N |
| 좌측 수신정보 | 팩스 | recipientFax | 상단 좌측 4행 | 12~14pt | 없으면 '-' | N |
| 우측 회사정보 | 회사로고 | logoImage | 상단 우측 | 원본 비율 유지 | 파일 누락시 텍스트 로고 | Y |
| 우측 회사정보 | 주소 | companyAddress | 로고 하단 | 11~12pt | 줄바꿈 허용 | Y |
| 우측 회사정보 | 전화/팩스 | companyPhone/companyFax | 주소 하단 | 11~12pt | 고정 포맷 | Y |
| 중단 제목 | 납품 내역 타이틀 | deliveryDate | 표 상단 중앙 | 14~16pt | 예: 2025년 12월 납품 내역 | Y |
| 품목표 | No | items[].itemNo | 표 1열 | 가운데 정렬 | 숫자 | Y |
| 품목표 | 제품명 | items[].productName | 표 2열 | 좌측 정렬 | 길면 줄바꿈 | Y |
| 품목표 | 규격 | items[].specification | 표 3열 | 가운데 정렬 | 없으면 공백 | N |
| 품목표 | 수량 | items[].quantity | 표 4열 | 우측 정렬 | #,##0 또는 #,##0.## | Y |
| 품목표 | 단가 | items[].unitPrice | 표 5열 | 우측 정렬 | #,##0 | Y |
| 품목표 | 금액 | items[].amount | 표 6열 | 우측 정렬 | #,##0 | Y |
| 합계박스 | 공급가액 | subtotal | 우하단 박스 1행 | Bold, 우측 정렬 | #,##0 | Y |
| 합계박스 | 부가세 | vatAmount | 우하단 박스 2행 | Bold, 우측 정렬 | #,##0 | Y |
| 합계박스 | 총금액 | totalAmount | 우하단 박스 3행 | Bold, 우측 정렬 | #,##0 | Y |
| 하단 메모 | 지불조건 | paymentTerms | 좌하단 메모영역 | 11~12pt | 없으면 '정기 결제' | N |
| 하단 메모 | 계좌번호 | bankAccount | 좌하단 메모영역 | 11~12pt | 없으면 기본 계좌 | N |
| 서명 | 인수자 서명(이름) | receiverName | 하단 좌측 | 12~14pt | 없으면 공백 라인 | N |
| 직인 | 도장 이미지 | stampImage | 하단 우측 | 90~140px | 없으면 빈 박스 | N |

## 4) 검증 체크리스트 (개발 전달용)

- [ ] PDF/Excel 모두 샘플과 동일한 컬럼 비율인지 확인
- [ ] 숫자 포맷(수량/단가/금액) 일치 확인
- [ ] 긴 제품명 줄바꿈 시 행높이/페이지 깨짐 여부 확인
- [ ] 빈값 처리 규칙('-', 공백, 기본값) 일치 확인
- [ ] 월합명세서 필터(다중카테고리/텍스트검색) 결과가 문서 출력 데이터와 동일한지 확인
